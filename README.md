## Installation

OS X & Linux:

```sh
git clone https://www.github.com/phillip-jaeslee/PULSIM
cd PULSIM
pip install . 
```

We strongly NOT recommend install via
```sh
python3 setup.py
```
This might not be available for uninstallation.
